# Wss-Pinly

A hyperlocal business discovery platform that helps users find trusted local businesses and services nearby.

## Tech Stack

### Frontend
- React.js

### Backend
- Node.js
- Express.js

### Database
- MySQL

## Project Structure

frontend/
backend/

## Setup

### Frontend
cd frontend/pinly
- npm install
- npm start

### Backend
cd backend
- npm install
- npm start
